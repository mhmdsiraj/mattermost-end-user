### أرشفة قناة 

أرشِف القنوات العامة والخاصة عندما لا تعود هناك حاجة إليها. الأرشفة تزيل القناة من الشريط الجانبي وتضعها في حالة قراءة فقط. أي عضو في القناة يمكنه أرشفتها ما لم يقم مسؤول النظام بتعطيل هذا الحق.

<Aside type="note" >
    يمكنك الاستمرار في الوصول إلى القنوات المؤرشفة ما لم يُمنع ذلك بواسطة إعدادات النظام.
 </Aside>


<Tabs>
  <TabItem label="الويب/سطح المكتب (Web/Desktop)" icon="laptop">
    لتأرشفة قناة: افتح قائمة القناة من اسم القناة في أعلى اللوحة المركزية ثم اختر **أرشفة القناة (Archive Channel)**.
  </TabItem>

  <TabItem label="الهاتف المحمول (Mobile)" icon="mobile">
    <Steps>
    1. اضغط على القناة التي تريد أرشفتها.
       <Image src={mobileSelectChannel} alt="Step 1" width={300} style={{ borderRadius: '15px', border: '1px solid #ddd', display: 'block', margin: '2rem auto' }} />

    2. اضغط أيقونة **المزيد (More)** في الزاوية العلوية اليمنى.
       <Image src={mobileMoreOptions} alt="Step 2" width={300} style={{ borderRadius: '15px', border: '1px solid #ddd', display: 'block', margin: '2rem auto' }} />

    3. اضغط **عرض المعلومات (View info)**.
       <Image src={mobileViewInfo} alt="Step 3" width={300} style={{ borderRadius: '15px', border: '1px solid #ddd', display: 'block', margin: '2rem auto' }} />

    4. اختر **أرشفة القناة (Archive Channel)** ثم أكد بالضغط على **نعم (Yes)**.
       <Image src={mobileEditChannel} alt="Step 4" width={300} style={{ borderRadius: '15px', border: '1px solid #ddd', display: 'block', margin: '2rem auto' }} />

    5. اضغط **نعم (Yes)** للتأكيد النهائي.
       <Image src={mobileConfirmArchive} alt="Step 5" width={300} style={{ borderRadius: '15px', border: '1px solid #ddd', display: 'block', margin: '2rem auto' }} />
    </Steps>
  </TabItem>
</Tabs>

<Aside type="note" >
    <ul style={{ listStyleType: 'disc', paddingInlineStart: '1.5rem' }}>
      <li>عند تعطيل مستخدم، قد تُؤرشف قنوات الرسائل المباشرة المرتبطة به وتصبح للقراءة فقط.</li>
      <li>لا يمكن أرشفة قنوات الرسائل الجماعية، ويمكن إغلاقها لإخفائها من الشريط الجانبي.</li>
      <li>لا يمكن أرشفة قناة Town Square الافتراضية.</li>
      <li>يمكن لمسؤولي النظام أرشفة القنوات عبر وحدة التحكم دون الانضمام إليها.</li>
      <li>لا يمكن إعادة استخدام عنوان URL لقناة مؤرشفة عند إنشاء قناة جديدة باستخدام نفس الاسم.</li>
    </ul>
 </Aside>

### إلغاء أرشفة قناة 


يمكن لمسؤولي النظام ومسؤولي الفريق استعادة القنوات المؤرشفة — يتم استعادة العضويات والمحتوى ما لم تحذف الرسائل أو الملفات بواسطة سياسة الاحتفاظ.

<Tabs>
  <TabItem label="الويب/سطح المكتب (Web/Desktop)" icon="laptop">
    <Steps>
    1. للعثور على القناة، ابحث عنها ثم افتحها.
    2. من قائمة القناة اختر **إلغاء أرشفة القناة (Unarchive Channel)**.
    </Steps>
    <Image src={unarchiveChannelImg} alt="Unarchive Channel Web" style={{ height: '450px', width: 'auto', borderRadius: '15px', border: '1px solid #ddd', display: 'block', margin: '2rem auto' }}/>
  </TabItem>

  <TabItem label="الهاتف المحمول (Mobile)" icon="mobile">
    <Steps>
    1. اضغط على القناة التي تريد إلغاء أرشفتها.
       <Image src={mobileSelectChannel} alt="Select Channel" width={300} style={{ borderRadius: '15px', border: '1px solid #ddd', display: 'block', margin: '2rem auto' }} />

    2. اضغط أيقونة **المزيد (More)**.
       <Image src={mobileMoreOptions} alt="More Options" width={300} style={{ borderRadius: '15px', border: '1px solid #ddd', display: 'block', margin: '2rem auto' }} />

    3. اضغط **عرض المعلومات (View info)**.
       <Image src={mobileViewInfo} alt="View Info" width={300} style={{ borderRadius: '15px', border: '1px solid #ddd', display: 'block', margin: '2rem auto' }} />

    4. اضغط **إلغاء أرشفة القناة (Unarchive Channel)**.
       <Image src={mobileUnarchive} alt="Unarchive Click" width={300} 
       style={{ borderRadius: '15px', border: '1px solid #ddd', display: 'block', margin: '2rem auto' }} />

    5. اضغط **نعم (Yes)** للتأكيد.
       <Image src={mobileConfirmUnarchive} alt="Confirm Unarchive" width={300}    style={{ borderRadius: '15px', border: '1px solid #ddd', display: 'block', margin: '2rem auto' }} />
    </Steps>
  </TabItem>
</Tabs>

<Aside type="note" >
    يمكن لمسؤولي النظام إلغاء أرشفة القنوات عبر `mmctl` والمسؤولون الآخرون عبر واجهة برمجة التطبيقات.
 </Aside>
