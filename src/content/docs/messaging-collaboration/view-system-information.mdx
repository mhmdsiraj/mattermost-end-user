## معلومات حوار "حول" (About dialog information)

يعرض حوار "حول" معلومات أساسية حول مثيل Mattermost الخاص بك، بما في ذلك:

- **إصدار Mattermost (Mattermost Version)**: الإصدار الحالي لخادم Mattermost. انقر لنسخ رقم الإصدار.
- **إصدار تطبيق سطح المكتب (Desktop App Version)**: الإصدار الحالي لتطبيق Mattermost لسطح المكتب (إن وجد). انقر لنسخ رقم الإصدار.
- **إصدار مخطط قاعدة البيانات (Database Schema Version)**: إصدار مخطط قاعدة البيانات المستخدم.
- **الترخيص (License)**: معلومات حول ترخيص Mattermost الخاص بك (إن وجد).
- **معلومات البناء (Build Information)**: تفاصيل حول بناء الخادم.
- **مقياس التحميل (Load Metric)**: المستخدمون النشطون شهريًا بالنسبة للمستخدمين المرخصين. راجع التفاصيل في وثائق الدعم.
