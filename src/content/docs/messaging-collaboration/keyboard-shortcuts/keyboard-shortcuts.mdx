keyboard when using Mattermost in a web browser or the desktop app.

  pressing `Ctrl` `/` on <span dir="ltr"><span dir="ltr"><span dir="ltr"><span dir="ltr"><span dir="ltr">Windows</span></span></span></span></span> or Linux, pressing `‚åò` `/` on <span dir="ltr"><span dir="ltr"><span dir="ltr"><span dir="ltr"><span dir="ltr">macOS</span></span></span></span></span>,
