تساعدك اختصارات لوحة المفاتيح (Keyboard shortcuts) في ماترموست على تحقيق أقصى استفادة من لوحة المفاتيح لزيادة كفاءة عملك، سواء كنت تستخدم ماترموست عبر متصفح الويب أو تطبيق سطح المكتب.

:::tip[نصائح إضافية]

- تتوفر اختصارات إضافية؛ يمكنك الاطلاع على توثيق [اختصارات لوحة مفاتيح الفريق](/end-user-guide/collaborate/team-keyboard-shortcuts) وتفاصيل [تنسيق النصوص](/end-user-guide/collaborate/format-messages:text-style).
- لعرض قائمة الاختصارات المتاحة داخل ماترموست، اضغط على `Ctrl` + `/` في ويندوز ولينكس، أو `⌘` + `/` في نظام macOS، أو استخدم الأمر المائل `/shortcuts`.

![مراجعة قائمة اختصارات لوحة المفاتيح المتاحة باستخدام الأمر المائل /shortcuts](https://docs.mattermost.com/_images/keyboard-shortcuts.png)
:::

## التنقل بين القنوات

تدعم المتصفحات المتوافقة وتطبيق ماترموست لسطح المكتب الاختصارات التالية للتنقل بين القنوات (Channels):

| الوصف                                                                                                       | على Mac                | على Windows و Linux      |
| :---------------------------------------------------------------------------------------------------------- | :--------------------- | :----------------------- |
| القناة أو الرسالة الخاصة السابقة في الشريط الجانبي.                                                         | `⌥` + `↑`              | `Alt` + `↑`              |
| القناة أو الرسالة الخاصة التالية في الشريط الجانبي.                                                         | `⌥` + `↓`              | `Alt` + `↓`              |
| القناة أو الرسالة الخاصة السابقة التي تحتوي على رسائل غير مقروءة.                                           | `⌥` + `⇧` + `↑`        | `Alt` + `Shift` + `↑`    |
| القناة أو الرسالة الخاصة التالية التي تحتوي على رسائل غير مقروءة.                                           | `⌥` + `⇧` + `↓`        | `Alt` + `Shift` + `↓`    |
| تحديد آخر منشور في القناة كـ "غير مقروء".                                                                   | `⌥` + النقر على القناة | `Alt` + النقر على القناة |
| - إذا لم يتم تحديد نص: فتح نافذة **البحث عن القنوات**.<br/>- إذا تم تحديد نص: إنشاء رابط تشعبي (Hyperlink). | `⌘` + `K`              | `Ctrl` + `K`             |
| فتح نافذة **الرسائل الخاصة** (Direct Messages).                                                             | `⌘` + `⇧` + `K`        | `Ctrl` + `Shift` + `K`   |
| فتح نافذة **الإعدادات** (Settings).                                                                         | `⌘` + `⇧` + `A`        | `Ctrl` + `Shift` + `A`   |
| فتح "الإشارات" (Mentions) الأخيرة.                                                                          | `⌘` + `⇧` + `M`        | `Ctrl` + `Shift` + `M`   |
| نقل التركيز إلى حقل إدخال الرسائل الرئيسي في القناة.                                                        | `⌘` + `⇧` + `L`        | `Ctrl` + `Shift` + `L`   |
| فتح أو إغلاق الشريط الجانبي الأيمن.                                                                         | `⌘` + `.`              | `Ctrl` + `.`             |
| توسيع أو تصغير عرض الشريط الجانبي الأيمن.                                                                   | `⌘` + `⇧` + `.`        | `Ctrl` + `Shift` + `.`   |
| نقل التركيز إلى حقل **البحث** والبحث في القناة الحالية.                                                     | `⌘` + `⇧` + `F`        | `Ctrl` + `Shift` + `F`   |
| البحث عن القنوات غير المقروءة أو البحث في كافة القنوات.                                                     | `⌘` + `⇧` + `U`        | `Ctrl` + `Shift` + `U`   |
| فتح أو إغلاق تفاصيل **معلومات القناة** (Channel Info).                                                      | `⌘` + `⇧` + `I`        | `Ctrl` + `Shift` + `I`   |

## رفع الملفات

يمكنك استخدام الاختصار التالي لرفع الملفات (File uploads) في كافة الأنظمة:

| الوصف    | على Mac   | على Windows و Linux |
| :------- | :-------- | :------------------ |
| رفع ملف. | `⌘` + `U` | `Ctrl` + `U`        |

## إدارة الرسائل

تدعم المتصفحات وتطبيق سطح المكتب الاختصارات التالية للتعامل مع الرسائل (Messages):

| الوصف                                                                  | على Mac                | على Windows و Linux    |
| :--------------------------------------------------------------------- | :--------------------- | :--------------------- |
| إعادة كتابة الرسالة السابقة أو الأمر المائل الذي أدخلته (في حقل فارغ). | `⌘` + `↑`              | `Ctrl` + `↑`           |
| إعادة كتابة الرسالة التالية أو الأمر المائل الذي أدخلته (في حقل فارغ). | `⌘` + `↓`              | `Ctrl` + `↓`           |
| الرد على أحدث رسالة في القناة الحالية (في حقل فارغ).                   | `⇧` + `↑`              | `Shift` + `↑`          |
| تعديل آخر رسالة أرسلتها في القناة الحالية (في حقل فارغ).               | `↑`                    | `↑`                    |
| الإكمال التلقائي لاسم المستخدم (username@).                            | `@[character]` + `Tab` | `@[character]` + `Tab` |
| الإكمال التلقائي لاسم القناة (~).                                      | `~[character]` + `Tab` | `~[character]` + `Tab` |
| الإكمال التلقائي للرموز التعبيرية (Emoji).                             | `:[character]` + `Tab` | `:[character]` + `Tab` |
| التفاعل (Reaction) مع آخر رسالة في القناة أو سلسلة المحادثات.          | `⌘` + `⇧` + `\`        | `Ctrl` + `Shift` + `\` |

## تنسيق الرسائل

استخدم هذه الاختصارات لتنسيق المحتوى (Message formatting) بسرعة:

| الوصف                     | على Mac         | على Windows و Linux  |
| :------------------------ | :-------------- | :------------------- |
| جعل النص عريضاً (Bold).   | `⌘` + `B`       | `Ctrl` + `B`         |
| جعل النص مائلاً (Italic). | `⌘` + `I`       | `Ctrl` + `I`         |
| تنسيق النص كرابط.         | `⌘` + `⌥` + `K` | `Ctrl` + `Alt` + `K` |

## التنقل وتسهيل الوصول

تساعد هذه الاختصارات على تحسين إمكانية الوصول (Accessibility) وسرعة التنقل:

| الوصف                                         | على Mac       | على Windows و Linux |
| :-------------------------------------------- | :------------ | :------------------ |
| الانتقال إلى القناة السابقة في سجل تصفحك.     | `⌘` + `[`     | `Alt` + `←`         |
| الانتقال إلى القناة التالية في سجل تصفحك.     | `⌘` + `]`     | `Alt` + `→`         |
| تظليل النص حتى السطر السابق (في حقل الإدخال). | `⇧` + `↑`     | `Shift` + `↑`       |
| تظليل النص حتى السطر التالي (في حقل الإدخال). | `⇧` + `↓`     | `Shift` + `↓`       |
| إنشاء سطر جديد.                               | `⇧` + `Enter` | `Shift` + `Enter`   |

:::note
على الرغم من دعم ماترموست للغات وتخطيطات لوحة المفاتيح القياسية، قد لا تعمل بعض الاختصارات إذا كنت تستخدم برامج تخصيص مفاتيح (Keymapping) تتعارض مع اختصارات المتصفح الافتراضية.
:::

## المكالمات (Calls)

عند تفعيل ميزة [المكالمات](/end-user-guide/collaborate/make-calls)، تتوفر الاختصارات التالية:

| الوصف                                           | على Mac                      | على Windows و Linux                   |
| :---------------------------------------------- | :--------------------------- | :------------------------------------ |
| بدء مكالمة أو الانضمام إليها في القناة الحالية. | `⌘` + `⌥` + `S`              | `Ctrl` + `Alt` + `S`                  |
| كتم الصوت أو إلغاء الكتم.                       | `⌘` + `⇧` + `Space`          | `Ctrl` + `Shift` + `Space`            |
| رفع اليد أو خفضها.                              | `⌘` + `⇧` + `Y`              | `Ctrl` + `Shift` + `Y`                |
| مشاركة الشاشة أو إيقاف مشاركتها.                | `⌘` + `⇧` + `E`              | `Ctrl` + `Shift` + `E`                |
| إظهار أو إخفاء قائمة المشاركين.                 | `⌥` + `P` أو `⌘` + `⇧` + `P` | `Alt` + `P` أو `Ctrl` + `Shift` + `P` |
| مغادرة المكالمة الحالية.                        | `⌘` + `⇧` + `L`              | `Ctrl` + `Shift` + `L`                |
| الضغط المستمر للتحدث (Push to talk).            | `Space`                      | `Space`                               |
